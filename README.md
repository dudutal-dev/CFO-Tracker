## 🚀 פיצ'רים עיקריים

### 📌 ניהול משרות
- 23 משרות **מוכנות מראש** מסריקה של אפריל 2026
- 4 רמות עדיפות (דחוף/גבוהה/בינונית/נמוכה) עם קוד צבעים
- 13 שלבי תהליך — מ"לא התחיל" ל"נחתם חוזה"
- 6 סטטוסים — פעיל/מוקפא/סגור (3 וריאציות)/לא רלוונטי
- Quick actions: קדם שלב, סמן בוצע, רישום יומן, קיבלתי!

### 🕵️ רדאר — שוק נסתר
- **8 ציידי ראשים** מוכנים (Spencer Stuart, Korn Ferry, Heidrick, וכו')
- **5 חברות יעד** מוכנות (Azrieli, Melisron, ZIM, וכו')
- **3 סיגנלים חמים** (רפאל, שופרסל, Ondas)
- ניהול **Networking** עם עמיתי CFO

### 📄 תיקיית CVs
- העלאת PDF/DOCX עד 2MB
- ריבוי גרסאות (עברית/אנגלית/ממוקד ציבורי/היי-טק)
- הורדה מחדש, שינוי שם, מחיקה

### 📧 שליחת מייל דרך Gmail
- בחר CV מהתיקייה → Gmail composer נפתח עם תבנית מוכנה
- אוטו-חילוץ כתובת מייל משדה "איש קשר"

### 🎯 פרופיל חיפוש אישי
- סוגי תפקידים מעניינים
- סקטורים מועדפים
- גיאוגרפיה
- טווח שכר
- הערות חופשיות

### 🔍 חיפוש באתרי משרות (חדש!)
- **20+ מקורות מוכנים** עם קישורי חיפוש ממוקדים ב-CFO:
  - 🇮🇱 **אתרי משרות ישראלים**: AllJobs, Drushim, JobMaster, JobNet, JobInfo, TheMarker Career, Jobify, BeBee
  - 💼 **LinkedIn**: CFO / VP Finance / Country CFO (פילטרים שונים)
  - 🌍 **בינלאומי**: Glassdoor, Indeed
  - 📡 **עיתונות + סיגנלים**: Calcalist מינויים, Globes, TheMarker, Bizportal, מאיה TASE
- **הוסף מקור משלך** (כל URL)
- **יומן חיפושים**: "נבדק: 20/04/2026 (×3)" — רואים מתי לאחרונה סקרת כל מקור
- **לחיצה אחת → פותח את האתר** עם חיפוש ממוקד CFO+ישראל

### 🔄 סנכרון עדכוני משרות
- ייבוא קובץ `cfo-sync-YYYY-MM-DD.json` מ-Claude
- **Smart merge**: משרות קיימות לא נוגעות (כולל ההערות והיומן שלך)
- רק משרות חדשות באמת מתווספות

### 💾 גיבוי מלא
- **ב-Chrome/Edge Desktop**: בוחר היכן לשמור (File System API)
- **במובייל**: יורד לתיקיית Downloads
- **פורמט JSON**: משרות + רדאר + פרופיל ב-snapshot אחד
- **CSV**: ייצוא משרות ל-Excel/Google Sheets

### ☁️ סנכרון Google Drive (חדש!)
- **גיבוי בלחיצה אחת** ל-Drive האישי שלך
- **שחזור מגיבוי** ברשימה של כל הגיבויים הקודמים
- **תיקייה ייחודית**: `CFO Tracker Backups` ב-Drive
- **לא שרת חיצוני** — הקובץ אצלך, פרטי לחלוטין
- דורש הגדרה חד-פעמית של Google Client ID — ראה `GOOGLE_DRIVE_SETUP.md`

### 🎨 עיצוב
- **Light mode** כברירת מחדל + **Dark mode** זמין
- **3 מצבי תצוגה**:
  - 📱 **מובייל** — מסך יחיד צר (640px)
  - 🖥️ **מחשב** — 2-עמודות, שימוש ברוחב (1400px)
  - 🔄 **אוטומטי** — מתאים לפי רוחב המסך
- נגישות WCAG AAA (ניגודיות 4.5:1+)

## 💾 היכן הנתונים שומרים?

הנתונים נשמרים **מקומית על המכשיר** (localStorage):

```
cfo_jobs_tracker_dudu_v1    → המשרות
cfo_radar_items             → רדאר (שוק נסתר)
cfo_search_profile          → פרופיל חיפוש
cfo_cvs_list                → רשימת CVs
cfo_cv_{id}                 → תוכן CV (base64)
cfo_tracker_theme           → light / dark
cfo_tracker_layout          → auto / mobile / desktop
```

**⚠️ חשוב**: אין סנכרון בין מכשירים. להעברה — השתמש בייצוא JSON ו-ייבוא במכשיר השני.
